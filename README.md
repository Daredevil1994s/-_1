# Проект "Ретротрасляция геотрекеров автомобилей в интересах транспортной компании с помощью API"

## Описание
Проект "Ретротрасляция геотрекеров автомобилей в интересах транспортной компании с помощью API" разрабатывается с целью обеспечить транспортную компанию инструментом для мониторинга и отслеживания автомобилей, основываясь на данных от геотрекеров.

Основной функционал проекта включает:
- Сбор данных о местоположении автомобилей с геотрекеров.
- Преобразование и обработка полученных данных.
- Ретротрасляцию местоположения автомобилей в виде удобного веб-интерфейса для пользователей компании.

## Функции
- Интеграция с геотрекерами автомобилей для сбора данных о местоположении источников.
- Преобразование и обработка сырых данных для извлечения полезной информации.
- Развертывание системы на сервере с возможностью доступа через веб-интерфейс.
- Административные функции для управления доступом пользователей и конфигурацией системы.

## Технологии
- Язык программирования: Python.
- Фреймворк: Fast API.
